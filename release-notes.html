Major changes and improvements for v2.0.0

v1.0.5

New features and improvements for v1.0.5

  • Added Search in Reports
  • Automatic Data Cleanup: The extension now automatically clears report data older than 30 days.
  • Redesigned Tracking Toggle: The main toggle in the popup has been redesigned. The NetMeter logo now acts as the button to enable or disable tracking for a site, creating a cleaner, more modern look.
  • Improved Reports UI
