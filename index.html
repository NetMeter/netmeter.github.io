Privacy

Privacy First

We don't track your data. Our extensions are designed to work locally and respect your browsing privacy. No remote logging, no hidden tracking.

Lightweight

Built with modern tech like Preact and TypeScript to ensure zero impact on your browser's performance. Fast loading and low memory footprint.

Developer Friendly

Deep scan tools and granular controls designed specifically for developers. Inspect storage values that standard devtools often miss.

Behind the Project

NetMeter is an independent project born out of a need for more precise and transparent browser tools. Unlike many bloated extensions, NetMeter focuses on doing one thing exceptionally well with a clean, modern interface.

Every line of code is crafted to be as efficient as possible, ensuring that your tools don't become the problem they're trying to solve.
